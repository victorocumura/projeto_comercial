# Projeto Comercial e Estoque

Sistema em Python para gerenciamento comercial com três funcionalidades:

1. **Comissões de Vendedores**  
   - Calcula comissão por venda:
     - < R$100 → sem comissão
     - R$100 a R$499,99 → 1%
     - ≥ R$500 → 5%
   - Exibe o total por vendedor.

2. **Movimentação de Estoque**  
   - Entrada ou saída de produtos.
   - Cada movimentação tem ID único e descrição.
   - Atualiza e mostra estoque final do produto.

3. **Cálculo de Juros**  
   - Considera valor e data de vencimento.
   - Multa de 2,5% ao dia de atraso.
   - Exibe juros acumulados até hoje.
